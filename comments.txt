Utilizar o Github Pages, por conta do trackeamento das branches uma
local com o nome de gh-pages e uma remota com o mesmo nome para que 
possa ser utilizada uma página estática(HTML, CSS, JS), hospedada
no github.

Alguns comandos utilizados:
git branch --track gh-pages origin/gh-pages
git push gh-pages gh-pages
git branch -t gh-pages origin/gh-pages
